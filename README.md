# udemy-clone
Udemy Clone Project  

Objective:  

Replicate the udemy website   

Website link: https://www.udemy.com/  

Requirements  

1. HTML Structure: Create semantic HTML for layout.  

2. Responsive Design: Use CSS (Flexbox/Grid) for an adaptable layout.  

3. Key Sections:  

    1. Navbar: Similar to Udemy's, with a logo, search bar, and navigation.  

    2. Menu Bar: Clone Udemy's menu.  

    3. Recommended Courses: Showcase courses with images and details.  

    4. Recommended Topics: Include an input box for topic suggestions.  

    5. Popular Courses: Display courses sorted by ratings and reviews.  

    6. Footer: Identical to Udemy's, with links and copyright information.  

  

Outcome  

A static, responsive Udemy website clone demonstrating front-end development skills.
